## Topic 105：Shell 與 Shell 腳本

### 105.1 自訂與使用 Shell 環境（權重：4）

#### 考試重點
- 設定環境變數（如 PATH）於登入時或開啟新 Shell 時
- 撰寫 Bash 函式以封裝常用指令
- 維護新使用者帳號的骨架目錄（skeleton directory）
- 設定指令搜尋路徑

#### Shell 類型

| 類型 | 說明 | 讀取的啟動檔 |
|------|------|------------|
| **互動式登入 Shell** | 使用者登入系統時執行 | `/etc/profile` → `~/.bash_profile` 或 `~/.bash_login` 或 `~/.profile` |
| **互動式非登入 Shell** | 登入後再開啟的終端 | `/etc/bash.bashrc` → `~/.bashrc` |
| **非互動式 Shell** | 自動化腳本，不需人工輸入 | 不讀取任何啟動檔（如 cron job）|
| **非互動式登入 Shell** | 極少見，幾乎無實際用途 | — |

#### 啟動檔優先順序
```
全域（Global）          本地（Local，優先）
/etc/profile      <    ~/.bash_profile
/etc/bash.bashrc  <    ~/.bashrc
```
**重點：本地設定覆蓋全域設定**

#### 啟動 Shell 的方式
```bash
bash -l / bash --login    # 啟動登入 Shell
bash -i                   # 啟動互動式 Shell
bash --noprofile          # 忽略所有啟動設定檔
bash --norc               # 忽略 ~/.bashrc
bash --rcfile <file>      # 使用指定檔案作為啟動設定
```

#### 重要設定檔
```
/etc/profile          # 全域登入 Shell 設定
/etc/profile.d/*      # 附加全域設定（模組化）
/etc/bash.bashrc      # 全域互動式 Shell 設定
~/.bash_profile       # 個人登入 Shell 設定（優先）
~/.bash_login         # 個人登入設定（次選）
~/.profile            # 個人設定（最後選）
~/.bashrc             # 個人互動式 Shell 設定
~/.bash_logout        # 登出時執行
/etc/skel/            # 骨架目錄，新使用者的預設家目錄模板
```

#### 變數操作
```bash
# 設定本地變數
MY_VAR="hello"

# 設定環境變數（匯出）
export MY_VAR="hello"
export PATH=$PATH:/new/path

# 查看所有環境變數
env
printenv
set          # 列出所有變數與函式

# 取消設定
unset MY_VAR

# 唯讀變數
readonly MY_VAR="fixed"

# 在修改環境下執行程式
env VAR=value command
```

#### 常見環境變數

| 變數 | 說明 |
|------|------|
| `PATH` | 指令搜尋路徑 |
| `HOME` | 使用者家目錄（`~` 的意思）|
| `USER` | 目前使用者名稱 |
| `SHELL` | 目前使用的 Shell |
| `LANG` | 語言設定 |
| `PS1` | 命令提示符格式 |
| `HISTSIZE` | 記憶歷史指令數量 |
| `HISTFILE` | 歷史指令儲存檔 |
| `DISPLAY` | X Window 顯示器設定 |

#### Alias（別名）
```bash
# 建立別名
alias ll='ls -la'
alias rm='rm -i'

# 查看所有別名
alias

# 移除別名
unalias ll

# 讓別名永久生效 → 寫入 ~/.bashrc
echo "alias ll='ls -la'" >> ~/.bashrc
source ~/.bashrc
```

#### 函式（Function）
```bash
# 定義函式
greet() {
    echo "Hello, $1!"
}
# 或
function greet {
    echo "Hello, $1!"
}

# 呼叫函式
greet World   # 輸出：Hello, World!

# 特殊變數
$0   # 腳本/函式名稱
$1~$9 # 位置參數
$#   # 參數數量
$@   # 所有參數
$?   # 上一指令的退出狀態（0=成功）
$$   # 目前 Shell 的 PID
$!   # 最後背景執行的 PID
```

### 105.2 撰寫簡單的 Shell 腳本（權重：4）

#### 腳本基本結構
```bash
#!/bin/bash
# 這是註解

# 執行腳本前需給予執行權限
chmod +x myscript.sh
./myscript.sh
```

#### 條件判斷
```bash
# if 語句
if [ $var -gt 10 ]; then
    echo "大於10"
elif [ $var -eq 10 ]; then
    echo "等於10"
else
    echo "小於10"
fi

# case 語句
case $var in
    "yes") echo "是" ;;
    "no")  echo "否" ;;
    *)     echo "其他" ;;
esac
```

#### 常用測試運算子

| 數字比較 | 字串比較 | 檔案測試 |
|---------|---------|---------|
| `-eq` 等於 | `=` 等於 | `-f` 是一般檔案 |
| `-ne` 不等於 | `!=` 不等於 | `-d` 是目錄 |
| `-gt` 大於 | `-z` 字串為空 | `-e` 存在 |
| `-lt` 小於 | `-n` 字串非空 | `-r` 可讀 |
| `-ge` 大於等於 | | `-w` 可寫 |
| `-le` 小於等於 | | `-x` 可執行 |

#### 迴圈
```bash
# for 迴圈
for i in 1 2 3; do
    echo $i
done

for file in *.txt; do
    echo $file
done

# while 迴圈
while [ $count -lt 5 ]; do
    echo $count
    count=$((count + 1))
done

# until 迴圈（直到條件成立才停）
until [ $count -ge 5 ]; do
    echo $count
    count=$((count + 1))
done
```

#### 算術運算
```bash
result=$((3 + 5))
result=$(expr 3 + 5)
```

## Topic 106：使用者介面與桌面環境

### 106.1 安裝與設定 X11（權重：2）

#### X Window System 架構
- **X Server**：控制實際顯示硬體（螢幕、鍵盤、滑鼠）
- **X Client**：應用程式，向 X Server 發送顯示請求
- **DISPLAY 變數**：格式為 `hostname:display.screen`，例如 `:0.0`

#### 重要設定檔
```
/etc/X11/xorg.conf          # 主要設定檔
/etc/X11/xorg.conf.d/       # 模組化設定目錄
/usr/share/X11/xorg.conf.d/ # 系統預設設定
```

#### 常用指令
```bash
xdpyinfo          # 顯示 X server 資訊
setxkbmap tw      # 設定鍵盤配置（X 環境下）
localectl         # 設定鍵盤配置（系統層級）
Xorg -configure   # 自動生成 xorg.conf
```

#### Wayland
- X11 的新一代替代協定
- 更安全、效能更好
- 目前許多發行版已預設使用

### 106.2 圖形桌面環境（權重：1）

| 桌面環境 | 特色 |

#### 遠端桌面協定

| 協定 | 說明 |
|------|------|
| **XDMCP** | X 本地顯示管理協定 |
| **VNC** | 跨平台遠端桌面，傳輸畫面 |

| **SPICE** | 虛擬化環境最佳化的遠端協定 |

### 106.3 無障礙功能（權重：1）

- **Sticky Keys**：按鍵黏著（輔助單手操作）

## Topic 107：系統管理任務

### 107.1 管理使用者與群組帳號（權重：5）

#### 使用者管理指令
```bash
# 新增使用者
useradd username
useradd -m -s /bin/bash -G sudo username   # -m建立家目錄 -s指定Shell -G加入群組

# 修改使用者
usermod -aG groupname username  # 附加到群組
usermod -s /bin/bash username   # 更改Shell
usermod -l newname oldname      # 改使用者名稱

# 刪除使用者
userdel username         # 只刪帳號
userdel -r username      # 連同家目錄一起刪

# 設定密碼
passwd username

# 密碼效期管理
chage -l username        # 查看密碼效期
chage -M 90 username     # 最長90天必須換密碼
chage -E 2025-12-31 username  # 帳號到期日
```

#### 群組管理指令
```bash
groupadd groupname          # 新增群組
groupmod -n newname oldname # 改群組名
groupdel groupname          # 刪除群組
```

#### 重要設定檔

| 檔案 | 內容 |
|------|------|
| `/etc/passwd` | 使用者基本資訊（格式：user:x:UID:GID:備註:家目錄:Shell）|
| `/etc/shadow` | 使用者加密密碼（只有root可讀）|
| `/etc/group` | 群組資訊 |
| `/etc/gshadow` | 群組加密密碼 |
| `/etc/skel/` | 骨架目錄（新使用者家目錄的模板）|
| `/etc/login.defs` | 帳號建立的預設設定 |

```bash
# 查詢使用者/群組資訊
getent passwd username
getent group groupname
```

### 107.2 排程工作自動化（權重：4）

#### Cron — 定期排程

**Crontab 格式：**
```
分 時 日 月 週 指令
*  *  *  *  *  command

# 範例
0 2 * * * /backup.sh          # 每天凌晨2點執行
30 8 * * 1 /weekly.sh          # 每週一早上8:30執行
*/5 * * * * /check.sh          # 每5分鐘執行一次
0 0 1 * * /monthly.sh          # 每月1日午夜執行
```

**特殊時間關鍵字：**

| 關鍵字 | 等同於 |
|--------|--------|
| `@reboot` | 開機時執行一次 |
| `@daily` | `0 0 * * *` |
| `@weekly` | `0 0 * * 0` |
| `@monthly` | `0 0 1 * *` |
| `@yearly` | `0 0 1 1 *` |

```bash
crontab -e          # 編輯目前使用者的 crontab
crontab -l          # 查看目前使用者的 crontab
crontab -r          # 刪除目前使用者的 crontab
crontab -u user -l  # 查看指定使用者的 crontab（需root）
```

**重要目錄與檔案：**
```
/etc/crontab          # 系統 crontab（含使用者欄位）
/etc/cron.d/          # 系統 crontab 片段目錄
/etc/cron.allow       # 允許使用 cron 的使用者（白名單）
/etc/cron.deny        # 禁止使用 cron 的使用者（黑名單）
/var/spool/cron/      # 使用者 crontab 儲存位置
```

#### at — 一次性排程
```bash
at 14:30              # 指定時間執行
at now + 2 hours      # 2小時後執行
at tomorrow           # 明天同一時間

atq                   # 查看待執行工作清單
atrm <job_id>         # 刪除工作

# 存取控制
/etc/at.allow         # 允許使用 at 的使用者
/etc/at.deny          # 禁止使用 at 的使用者
```

#### systemd Timer（現代替代方案）
```bash
systemctl list-timers          # 查看所有計時器
systemd-run --on-active=2h command  # 2小時後執行一次（替代 at）
```

## Topic 108：核心系統服務

### 108.1 維護系統時間（權重：3）

#### 時鐘類型
- **系統時鐘（Software Clock）**：由 kernel 維護，存在 RAM 中
- **硬體時鐘（RTC/BIOS Clock）**：儲存在主機板，電源關閉後仍保存

```bash
date                              # 顯示目前時間
date -u                           # 顯示 UTC 時間
date +%s                          # 顯示 Epoch 時間（秒）
date -s "2024-01-01 12:00:00"     # 手動設定系統時間

hwclock                           # 顯示硬體時鐘
hwclock --systohc                 # 系統時鐘 → 硬體時鐘
hwclock --hctosys                 # 硬體時鐘 → 系統時鐘

timedatectl                       # 查看完整時間資訊
timedatectl set-time "12:00:00"   # 設定時間
timedatectl set-ntp true          # 啟用 NTP 同步
```

#### NTP 網路時間同步

**重要名詞：**

| 名詞 | 說明 |
|------|------|
| **Stratum** | 距離參考時鐘的層數（0=GPS等，1=直連，數字越小越準）|
| **Offset** | 系統時間與網路時間的差值 |
| **Jitter** | 自上次同步後的時間偏差量 |
| **Pool** | 共享負載的 NTP 伺服器群組 |

```bash
# ntpd（傳統 NTP）
ntpq -p             # 顯示 NTP peer 狀態
ntpdate <server>    # 手動單次同步

# chrony（現代替代）
chronyc tracking    # 顯示 NTP 同步狀態
chronyc sources     # 顯示 NTP 來源
chronyc ntpdata     # 顯示上次輪詢資訊
chronyc makestep    # 手動強制同步
```

### 108.2 系統日誌（權重：3）

#### rsyslog — 傳統日誌系統

**重要日誌目錄：**
```
/var/log/syslog 或 /var/log/messages  # 主要系統日誌
/var/log/auth.log 或 /var/log/secure  # 認證/登入日誌
/var/log/kern.log                      # 核心日誌
/var/log/cron                          # Cron 工作日誌
/var/log/mail.log                      # 郵件日誌
/var/log/dmesg                         # 開機訊息
```

**日誌優先級（嚴重性由高至低）：**

| 等級 | 數值 | 說明 |
|------|------|------|
| emerg | 0 | 系統無法使用 |
| alert | 1 | 需立即採取行動 |
| crit | 2 | 嚴重狀況 |
| err | 3 | 錯誤狀況 |
| warning | 4 | 警告狀況 |
| notice | 5 | 正常但值得注意 |
| info | 6 | 資訊訊息 |
| debug | 7 | 偵錯訊息 |

```bash
# 手動寫入日誌
logger "This is a test message"
logger -p auth.warning "Failed login attempt"

# 設定檔
/etc/rsyslog.conf
/etc/rsyslog.d/
```

#### journald — systemd 日誌系統

```bash
journalctl                           # 查看所有日誌
journalctl -f                        # 即時跟蹤日誌（類似 tail -f）
journalctl -n 50                     # 最後50行
journalctl -u sshd                   # 查看特定服務日誌
journalctl -p err                    # 只顯示錯誤等級以上
journalctl --since "2024-01-01"      # 指定時間後的日誌
journalctl --since "1 hour ago"      # 最近1小時
journalctl -b                        # 本次開機後的日誌
journalctl -b -1                     # 上次開機的日誌
journalctl --disk-usage              # 查看日誌占用磁碟空間
journalctl --vacuum-size=100M        # 清理日誌至100M以內

# 設定檔
/etc/systemd/journald.conf
```

### 108.3 郵件傳輸代理（MTA）基礎（權重：3）

```bash
/etc/aliases          # 系統郵件別名設定
newaliases            # 重新讀取 /etc/aliases（必須執行）
~/.forward            # 個人郵件轉寄設定
```

## Topic 109：網路基礎

### 109.1 網際網路協定基礎（權重：4）

#### IPv4 基礎

**IP 位址分類：**

| 類別 | 範圍 | 私有 IP 範圍 |
|------|------|------------|
| A | 1.0.0.0 ~ 126.255.255.255 | 10.0.0.0/8 |
| B | 128.0.0.0 ~ 191.255.255.255 | 172.16.0.0/12 |
| C | 192.0.0.0 ~ 223.255.255.255 | 192.168.0.0/16 |

**子網路遮罩（Netmask）：**
```
/24 = 255.255.255.0   → 254 台主機
/16 = 255.255.0.0     → 65534 台主機
/8  = 255.0.0.0       → 16777214 台主機
```

#### 重要 Port 號

| Port | 協定 | 服務 |
|------|------|------|
| 21 | TCP | FTP |
| 22 | TCP | SSH |
| 23 | TCP | Telnet |
| 25 | TCP | SMTP |
| 53 | TCP/UDP | DNS |
| 80 | TCP | HTTP |
| 110 | TCP | POP3 |
| 143 | TCP | IMAP |
| 443 | TCP | HTTPS |
| 631 | TCP | CUPS |

#### 傳輸協定比較

| | TCP | UDP |
|-|-----|-----|
| 連線 | 需要三向交握 | 無連線 |
| 可靠性 | 保證送達 | 不保證 |
| 速度 | 較慢 | 較快 |
| 用途 | HTTP、SSH、FTP | DNS、DHCP、串流 |

#### IPv6 基礎
- 128 位元位址，以冒號分隔的16進位表示
- 連續的 0 可縮寫：`2001:0db8::1`
- `::1` = localhost（IPv6）
- `fe80::/10` = Link-local 位址

### 109.2 持久性網路設定（權重：4）

#### 網路介面命名規則
- 傳統：`eth0`、`eth1`
- 新式（Predictable）：`enp3s0`、`wlan0`、`ens33`

```bash
# 查看網路介面
ip link show
ip addr show
nmcli device status

# 設定 IP（暫時）
ip addr add 192.168.1.10/24 dev eth0
ip link set eth0 up

# 路由操作
ip route show
ip route add default via 192.168.1.1

# 介面啟動/停止（舊式）
ifup eth0
```

### 110.1 執行安全管理任務（權重：3）

### 110.3 使用加密保護資料（權重：4）

#### OpenSSH

```bash
# 連線到遠端主機
ssh username@hostname
ssh -p 2222 username@hostname     # 指定 port

# 遠端執行指令
ssh user@host "ls -la"

# 金鑰管理
ssh-keygen -t ed25519             # 產生 ED25519 金鑰對（推薦）
ssh-keygen -t rsa -b 4096         # 產生 RSA 4096 位元金鑰

# 複製公鑰到遠端主機（啟用無密碼登入）
ssh-copy-id username@hostname

# SSH Agent（管理私鑰）
eval $(ssh-agent)                 # 啟動 SSH Agent
ssh-add ~/.ssh/id_ed25519         # 加入私鑰

# Port Tunnel（重要！）
ssh -L 8080:localhost:80 user@remote    # 本地轉發
ssh -R 8080:localhost:80 user@remote    # 遠端轉發
ssh -X user@remote                      # X11 轉發（遠端圖形應用）
```

**SSH 金鑰演算法：**

| 演算法 | 安全性 | 推薦度 |
|--------|--------|--------|
| ed25519 | 最強 | ⭐⭐⭐ 推薦 |
| ecdsa | 強 | ⭐⭐ |
| rsa | 需4096bit | ⭐ 可用 |
| dsa | 已不安全 | ❌ 不推薦 |

**重要設定檔：**
```
~/.ssh/id_ed25519         # 私鑰
~/.ssh/id_ed25519.pub     # 公鑰
~/.ssh/authorized_keys    # 允許登入的公鑰清單
~/.ssh/known_hosts        # 已知主機的公鑰記錄
/etc/ssh/sshd_config      # SSH Server 設定
/etc/ssh/ssh_config       # SSH Client 設定
```

#### GnuPG（GPG）— 檔案加密與數位簽章

```bash
# 產生金鑰對
gpg --gen-key
gpg --full-generate-key             # 完整選項

# 查看金鑰
gpg --list-keys                     # 公鑰清單
gpg --list-secret-keys             # 私鑰清單

# 匯出/匯入公鑰
gpg --export -a "User Name" > public.key    # 匯出公鑰
gpg --import public.key                      # 匯入公鑰

# 金鑰伺服器
gpg --keyserver keyserver.ubuntu.com --send-keys KEY_ID    # 上傳到金鑰伺服器
gpg --keyserver keyserver.ubuntu.com --recv-keys KEY_ID    # 從金鑰伺服器下載

# 加密與解密
gpg -e -r "recipient@email.com" file.txt    # 加密檔案
gpg -d file.txt.gpg                          # 解密檔案

# 數位簽章
gpg --sign file.txt                          # 簽署（二進位）
gpg --clearsign file.txt                     # 簽署（明文）
gpg --verify file.txt.asc                   # 驗證簽章

# 撤銷金鑰
gpg --gen-revoke KEY_ID                      # 產生撤銷憑證
```
